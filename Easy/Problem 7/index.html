JAVASCRIPT PROBLEM

In this lab, you will work with the Fetch API to retrieve some data from a provided URL and then display it on the web page. You will create a button to trigger the fetch request and a container to display the retrieved data. Moreover, you will also handle any errors encountered while fetching the data. Important Concepts: The Fetch API provides a JavaScript interface for fetching resources across the network. The fetch() function returns a Promise that resolves to the Response object representing the response to the request. To extract the JSON data from the response, you can use the response.json() method, which also returns a Promise. Promises can be chained using 'then' and 'catch' methods. Remember to handle errors that may arise during the fetching process.

Create a button with id 'fetchBtn' and a container with id 'dataContainer' to display fetched data. Attach a click event handler to the button and fetch data from the given URL: 'https://jsonplaceholder.typicode.com/posts/1'. Parse the fetched data as JSON and display the 'title' property in the 'dataContainer' as a heading (h3). Handle errors in the fetch request and display an error message inside the 'dataContainer'.
