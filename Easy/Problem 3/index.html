JAVASCRIPT PROBLEM

In this lab, you will be working with selectors in HTML and CSS. You will create a simple HTML document with a title and paragraph elements, and use CSS to style the elements based on their class name. Selectors are used in CSS to target specific elements in the HTML document and apply styles to them. The most commonly used selectors are element selectors (like 'p'), class selectors (like '.info'), and id selectors (like '#container'). In this lab, you will be using class and id selectors to style the elements.

Create a basic HTML skeleton with a title 'Selectors Lab'. Create a div with id 'container' and add 3 paragraphs inside it. Add the class 'info' to all the paragraphs. Use CSS to change the color of paragraphs with class 'info' to blue.
