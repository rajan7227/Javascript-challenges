JAVASCRIPT PROBLEM

In this lab, you'll be practicing how to work with environment variables in a Node.js application using the process.env object. You will explore how to read existing environment variables, as well as how to create new ones.

Create and export a helloWorld function from index.js Assign the value of the environment variable GREETING to a constant greeting Update the helloWorld function to return ${greeting} World where greeting is the value of the GREETING environment variable Create and export a setEnvironmentVariable function that sets a new environment variable
